# User Stories

## Meal Tracking

- As a user, I want to **take a photo of my meal** so that I remember what I ate.
- As a user, I want to **add a short description** (e.g., "chicken salad + rice").
- As a user, I want the app to **send the photo + text to AI** so that it estimates calories/macros.
- As a user, I want to **see the meal and its calories** in my daily log.

## Exercise Tracking

- As a user, I want to **take a screenshot of my workout summary**.
- As a user, I want to **add optional context** (e.g., "5k run").
- As a user, I want the app to **send screenshot + text to AI** so that it estimates calories burned.
- As a user, I want the **exercise calories added back into my daily budget**.

## Daily Budget

- As a user, I want to **set a daily calorie budget** (default 1500 kcal).
- As a user, I want to **see today’s budget, meals, exercise, and net balance**.
- As a user, I want to **check quickly whether I’m within my budget**.

## Configuration

- As a user, I want to **enter my OpenAI API key** so I can use AI features.
- As a user, I want to **save my daily calorie budget**.
- As a user, I want to **change these settings later**.

## Non-goals

- No weight tracking
- No exporting/sharing
- No social features
- No charts or analytics beyond daily totals
