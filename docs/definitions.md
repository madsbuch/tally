# Definitions

This page contains definitions. Please adhere to that everywhere in the codebase.

- Consumed Calories: Sum of entries with positive calory amount. Understood as sum of calories form meals
- Burned calories: Sum of entries with negative calory amount. Understood as sum of calories from activities
- Net Calories: Consumed calories subtracted burned calories
- Target Calories: The mount of calories the user is targeting for net calories in a time period
- Calories left: The difference between the net calories and the target calories for a period
